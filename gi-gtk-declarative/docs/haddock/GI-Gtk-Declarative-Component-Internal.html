class Typeable c => Component (c :: * -> *) where #

A declarative component. Types in this class must have a single type parameter which describes the type of the events emitted by the component.

Minimal complete definition

createComponent, update, view

Associated Types

data ComponentState c #

The internal state of the component. This is initialized when the component is first shown, and then varies over the lifetime of the component as patchComponent and update modify it.

data ComponentAction c #

Internal actions that are emitted by child widgets and make this component do stuff.

Methods

createComponent :: c event -> (ComponentState c, Maybe (ComponentAction c)) #

Called when the component is first shown. This creates the initial component state from the declarative component, and optionally provides an initial action to start the component doing stuff.

patchComponent :: ComponentState c -> c event -> ComponentState c #

Called when a new declarative component is applied to the existing component state. This method should copy any required state from the new declarative component. This method will never be called for top level components - so there is a default implementation that returns an error: components intended to be used a non-top-level components should implement this method.

update :: c event -> ComponentAction c -> UpdateM c event () #

Called when an action is emitted by a child event. The UpdateM monad allows updating the component state, running asynchronous IO actions, and sending events to the parent on this component.

view :: c event -> ComponentState c -> Widget (ComponentAction c) #

Provides the tree of widgets that display this component. Called whenever the state changes (and possibly other times too).

data UpdateM c event a where #

An "update action" that happens when a component of type c receives an action from a child widget. The event type parameter describes the type of events that the parent widget can receive. This implements MonadState to allow getting/setting the component state.
